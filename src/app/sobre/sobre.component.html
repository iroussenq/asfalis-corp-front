AsfalisCorp

------------------------ASFALIS-CORP------------------------ AsfalisCorp™(2022) é um programa feito com o intuito de treinar o uso dos frameworks Angular e Spring,utilizando SCSS e HTML e as linguagens de programação Java e Typescript. O proposito do programa é auxiliar o cadastro e reportagem de acidentes para policiais militares e federais simulando uma central(Banco de Dados Postgresql)
